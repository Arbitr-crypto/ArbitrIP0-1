# Arbi-Bot

Минимальный Telegram-бот для арбитража.

## Настройка
1. Вставьте токен бота в `bot.py` (`TELEGRAM_TOKEN`).
2. Вставьте ваш Telegram ID (`OWNER_CHAT_ID`) и ID оператора (`OPERATOR_ID`).

## Команды
- `/start` — показать кнопки управления.
- `/add_user <tg_id>` — добавить пользователя в whitelist.
- `/remove_user <tg_id>` — удалить пользователя из whitelist.
- `/list_users` — показать текущий whitelist.

## Кнопки
- **Старт сканера** — запускает отправку сигналов.
- **Стоп сканера** — останавливает сканер.
- **Поддержка** — ссылка на @Arbitr_IP
